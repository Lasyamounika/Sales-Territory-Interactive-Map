# Sales-Territory-Interactive-Map
Tableau dashboard for analyzing sales performance across Indian territories

## Project Overview
This project visualizes sales performance across Indian regions (North, South, East, West India) using an interactive Tableau dashboard.

## Features
- Geographic sales territory mapping
- Sales performance tiers (High, Medium, Low)
- Interactive date and performance filters
- Clear regional comparison

## Tools Used
- Tableau Desktop Public Edition
- Sales dataset (CSV/Excel)
- Mapbox / OpenStreetMap
